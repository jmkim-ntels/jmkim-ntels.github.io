## 🛠 2. 상세 셋업 가이드 (Setup Deep-Dive)

라이브에서 강조된 셋업의 핵심은 **'안정적인 에이전트 환경 구축'**입니다.

### 2.1 openclaw.json 최적화 설정
업데이트된 설정 파일에서 반드시 확인해야 할 항목들입니다.
- **`allowFiles: true`**: 에이전트가 로컬 파일에 접근하여 분석할 수 있도록 허용합니다. (기본값 false 권장이나 업무 지원 시 필수)
- **`pdfMaxBytesMb` & `pdfMaxPages`**: 대용량 기술 문서를 처리할 때 메모리 부족을 방지하기 위한 임계값 설정.

### 2.2 보안 강화: SecretRef 활용법
이제 `openclaw.json`에 직접 API 키를 적지 마세요!
- **작동 원리**: 환경 변수나 외부 암호화 저장소(`op` 등)를 참조하는 `SecretRef` 구조를 통해 설정 파일 노출 시에도 실제 자격 증명은 보호됩니다.
- **라이브 팁**: `openclaw configure --section secrets` 명령어를 통해 안전하게 관리하세요.
